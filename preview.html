STM32F103C8T6 Datasheet

基于 ARM Cortex-M3 内核的高性能微控制器，采用 LQFP48 封装。

参数 值 单位
内核 ARM Cortex-M3 -
主频 72 MHz
Flash 64 KB
RAM 20 KB
工作电压 2.0-3.6 V

代码示例

// LED 闪烁示例 #include "stm32f1xx.h" int main(void) { RCC->APB2ENR |= RCC_APB2ENR_IOPAEN; GPIOA->CRL &= ~GPIO_CRL_CNF5; GPIOA->CRL |= GPIO_CRL_MODE5_0; while(1) { GPIOA->ODR ^= GPIO_ODR_ODR5; for(int i = 0; i < 1000000; i++); } }
